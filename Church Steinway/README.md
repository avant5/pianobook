#### Recorded by Richard Luke ####
See Pianobook page for more details  
https://www.pianobook.co.uk/library/church-steinway

To hear this instrument in the SFZ format, visit:  
https://soundcloud.com/avant5/song-of-moonlight

### Installation ###

Download the EXS instrument from the Pianobook library and extract to your local computer.  Add the SFZ file to the **Instruments** directory and then load this file into your SFZ player.  

This version was built for and tested with the Sforzando player by Plogue, and not tested with other SFZ players.  

https://www.plogue.com/products/sforzando.html
